This repository has the code for the site lebooks.it which is used book marketplace where students of uni padova can list and buy books filtering through their coarses. The tech stack for frontend is
React(vite) + Typescript with Vanila CSS , backend- Python - FastAPI + SQLAlchemy backend, PostgreSQL schema design, JWT auth, and Alembic migrations 
azure communications services for emails - Containerized and deployed with Docker Compose on a Hetzner VM using Caddy reverse proxy, HTTPS, security
headers, and SPA routing.

Features - book listing and deleting , searching for books with department , program and coarse filter , contacting the seller with WebSockets, read receipts, and offline email notifications
This is  purly a  hobby project
